##최단경로
- 최단경로 정의
  - 간선의 가중치가 있는 그래프에서 두 정점 사이의 경로들 중에 간선의 가중치의 합이 최소인 경로
  - (간선의 가중치 없는 그래프 최단경로 -> 거쳐오는 간선수 최단 -> BFS)
- 하나의 시작 정점에서 끝 정점까지의 최단 경로
  - 다익스트라 알고리즘
    - 음의 가중치를 허용하지 않음
  - 벨만-포드 알고리즘
    - 음의 가중치를 허용
    - 간선을 중심으로 소비하는 알고리즘
- 모든 정점에서 모든 정점까지의 최단 거리
  - 플로이드-워샬 알고리즘
  <br><br>
  


###다익스트라(Dijkstra)
1. 출발 노드를 설정
2. 출발 노드를 기준으로 각 노드의 거리를 저장 (최단 거리 테이블을 초기화)
3. 방문하지 않은 노드 중에서 가장 거리가 짧은 노드를 선택
4. 해당 노드를 거쳐 다른 노드로 가는 비용을 계산하여 최단 거리 테이블을 갱신한다.
5. 3,4과정을 반복한다.



###플로이드워샬 (Floyd-Warshall)
- 모든 정점에서 모든 정점으로의 최단거리
- '거쳐가는 정점'을 기준으로 최단거리를 구한다.
- 연산 횟수는 V^3이고, 시간복잡도는 O(V^3)이다.

###벨만-포드 (Bellman-Ford-Moore)
- V개의 정점과 E개의 간선을 가진 가중 그래프 G에서 특정 출발 정점(S)에서부터 다른 모든 정점까지의 최단경로를 구하는 알고리즘
- 총 연산 횟수는 V*E이고, 시간복잡도는 O(VE)이다.
